# Bank Loan Dashboard (Power BI)

## 📊 Project Overview
This project focuses on analyzing bank loan data to understand loan approval trends, risk factors, and customer behavior.

## 🚀 Features
- Interactive Power BI dashboard
- Loan approval and rejection analysis
- Risk segmentation of customers
- KPI tracking (loan amount, approval rate, etc.)

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization

## 📈 Key Insights
- Identified patterns in loan approvals
- Highlighted high-risk customer segments
- Improved decision-making using dashboards
